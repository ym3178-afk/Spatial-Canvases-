Letters
Across
the Sea

A computational installation about family memory, paper, wind, flowers, distance, and return. This final version contains one static 2D canvas, two dynamic 2D canvases, and three 3D canvases.

2D Canvas 01 · Static

The Letter Archive

A quiet archive board: old paper, Thailand map, family photo, red thread, tickets, and memory labels.

2D Canvas 02 · Dynamic

Wishing Tree

Red flowers bloom on a tree. Hanging letters swing with wind. Move the mouse to change the wind.

2D Canvas 03 · Dynamic

Red Cotton Flower

A large red flower carries floating letter papers against a muted teal paper background.

3D Canvas 01 · Three.js

Invisible Routes

Hundreds of paper fragments form a constellation. The cursor acts like gravity.

3D Canvas 02 · Three.js

Sea of Letters

A field of papers becomes a moving ocean surface.

3D Canvas 03 · Three.js

Memory Shrine

A cinematic spatial shrine: paper fragments form a rotating altar of light, fog, red seals, and suspended memory.

Design Statement

This project uses code to turn memory into space. Paper becomes an archive, a tree, a flower, a constellation, an ocean, and a lantern. The interaction is intentionally quiet: wind, gravity, fading, and drifting replace buttons and menus.

Some letters crossed the sea.
Some never arrived.
